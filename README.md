# MERN Expense Tracker App with Chart.js

This is a full-stack web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js) for tracking expenses, with data visualization powered by Chart.js.

## Features

- **Expense Tracking**: Users can add, edit, and delete expenses.
- **Data Visualization**: The application provides visual representation of expense data using Chart.js.
- **Authentication**: Users can sign up, log in, and log out securely.
- **Persistent Data**: Expense data is stored in a MongoDB database, ensuring persistence across sessions.

## Technologies Used

- **MongoDB**: NoSQL database used for storing expense data.
- **Express.js**: Node.js web application framework used for building the backend API.
- **React.js**: JavaScript library used for building the user interface.
- **Node.js**: JavaScript runtime used for server-side logic.
- **Chart.js**: JavaScript library for data visualization.
